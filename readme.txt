Create beautiful WooCommerce product showcases using grid, list, slider, and table layouts. Compatible with Elementor and the WordPress block editor.

== Description ==

**Hexa Grid Product Showcase for WooCommerce** allows you to display your WooCommerce products in multiple modern layouts, including Grid, List, Slider, and Table. It is designed for store owners who want full control over how products are presented without writing any code.

The plugin works seamlessly with both Elementor and the WordPress Block Editor (Gutenberg), making it easy to build product displays anywhere on your site—pages, posts, or landing pages.

With flexible layout options and simple configuration, you can highlight featured products, best sellers, or custom product selections to improve user experience and increase conversions.

Key Features:

*   **3 Professional Layouts:** Grid, List, and Slider (Carousel).
*   **Showcase Builder:** Create unlimited presets via a clean Admin interface.
*   **Custom Post Type:** Manage your showcases just like posts.
*   **Flexible Query Builder:** Filter products by category, specific IDs, limit, and order.
*   **Fully Responsive:** Optimized for all device sizes.
*   **Developer Friendly:** Built with modern PHP standards (PSR-4).

== Installation ==

1.  Upload the `hexa-grid-product-showcase` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Go to **Product Showcase** in the admin menu to start creating your layouts.

== Frequently Asked Questions ==

= Does this plugin require WooCommerce? =
Yes, you must have WooCommerce installed and active.

= Can I filter products by category? =
Yes, you can specify a category slug in the Builder settings or use the `category="slug"` attribute in the shortcode.

= How do I create a slider? =
Simply create a new Preset, select "Slider" as the layout type, publish it, and use the generated shortcode.
